# 模块一：用户账户与认证模块 
    核心功能：处理所有与用户身份相关的事务。

    包含内容：

    后端：用户注册/登录API、JWT令牌生成与验证、密码加密（BCrypt）、个人信息管理。

    前端：登录页、注册页、个人中心页面（修改资料、管理地址）。

    数据库：用户表、收货地址表。

    特点：基础且关键，是其他模块的依赖。

# 模块二：菜单与商品管理模块
    核心功能：管理餐厅的核心产品——菜品。

    包含内容：

    后端：菜品的增、删、改、查API；菜品分类管理；设置菜品状态（上架/下架）。

    前端：

    用户端：菜单展示页面（按分类、搜索）、菜品详情。

    管理端：菜单管理页面（可操作所有菜品）。

    数据库：菜品分类表、菜品详情表。

    特点：数据驱动，以CRUD操作为主。
